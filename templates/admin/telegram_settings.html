Panduan Setup
1. Membuat Bot Telegram
  1. Chat @BotFather di Telegram
  2. Ketik /newbot
  3. Ikuti instruksi untuk membuat bot
  4. Salin token yang diberikan
2. Mendapatkan Group ID
  1. Tambahkan bot ke grup Telegram
  2. Kirim pesan /start di grup
  3. Gunakan bot @userinfobot untuk mendapatkan Group ID
  4. Atau forward pesan dari grup ke @userinfobot
